Option 2 - Complete Reset:
  rm vibesync.db && ./setup-database.sh && ./FINAL_START.sh

Option 3 - Check Backend:
  curl http://localhost:3000/health


┌──────────────────────────────────────────────────────────────┐
│  COMMON ERRORS                                               │
└──────────────────────────────────────────────────────────────┘

❌ "JSON Parse error"
   → Backend not running
   → Fix: bun run backend/server-improved.ts

❌ "Port 3000 in use"
   → Kill process: pkill -f backend

❌ "Database error"
   → Reset: rm vibesync.db && ./setup-database.sh
